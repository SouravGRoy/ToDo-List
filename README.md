A straightforward Todo List application built with React, allowing users to manage tasks efficiently by adding, viewing, and removing them.

## Introduction

This application offers a simple and intuitive interface for managing tasks. Users can easily add tasks by providing titles and descriptions, view the list of tasks, and remove them collectively when needed.

## Features

- **Add Tasks:** Input task titles and descriptions for quick task addition.
- **View Tasks:** Display the list of tasks with their respective titles and descriptions.
- **Remove Tasks:** Remove all tasks from the list at once for a clean slate.

## Installation

Follow these steps to run the application locally:

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/react-todo-list.git
   ```

2. Navigate to the project directory:

   ```bash
   cd react-todo-list
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

## Usage

Run the application:

```bash
npm start
```

The app will run in the development mode at `http://localhost:3000`.
